# Xendit Medusa Monorepo

A monorepo containing Xendit payment integration for Medusa.

## Packages

- `packages/xendit-medusa-plugin` - Xendit payment plugin for Medusa
- `packages/medusa-app` - Fullstack Medusa application with Next.js storefront

## Setup

```bash
# Install dependencies
pnpm install

# Run development servers
pnpm dev

# Build all packages
pnpm build

# Lint and format
pnpm lint
pnpm format
pnpm check
```

## Requirements

- Node.js >= 18
- pnpm >= 8
